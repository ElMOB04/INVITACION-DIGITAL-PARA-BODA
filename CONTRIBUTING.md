# Cómo contribuir

Gracias por querer ayudar. Aquí explico, de forma sencilla y humana, cómo colaborar con este proyecto.

1. Haz un fork del repositorio y crea una rama para tu cambio:

```bash
git checkout -b feat/mi-cambio
```

2. Haz cambios pequeños y claros. Si modificas estilos o scripts, indícalo en el commit.

3. Asegúrate de no incluir datos sensibles en tus commits (contraseñas, números personales, claves API).

4. Crea un Pull Request describiendo qué cambiaste y por qué. Si es una mejora visual, incluye capturas (`assets/`).

Estilo y buenas prácticas
- Usa un lenguaje claro en los mensajes de commit.
- Mantén HTML semántico y CSS organizado.
- Evita cambios masivos en un solo PR; divide en pasos si es necesario.

Contacto

Si tienes dudas escríbeme un issue y responderé lo antes posible.
